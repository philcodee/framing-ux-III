### Field consolidation (post-API processing)
Implemented three post-enrichment consolidation functions called after `groupDateFields()`:

**`groupPhoneFields()`**
- Detects consecutive text fields where first matches area code pattern and second matches phone/number pattern
- Merges into a single `phone-group` type with `(XXX) XXX-XXXX` formatted input
- Also handles 3-part phone (area + exchange + line)
- Parts written back individually to PDF on export

**`consolidateCheckboxSiblings()`**
- Groups consecutive checkbox fields that share a dotted ID parent prefix (e.g. `VehicleType.Sedan`, `VehicleType.Truck`)
- Merges into a single `checkbox-group` type rendered as a set of labeled checkboxes
- Strips the group name prefix from individual option labels for clean display
- Each part written as `'Yes'`/`'Off'` on PDF export

**`deduplicateByLabel()`**
- Normalizes labels (lowercase, collapse whitespace) and drops fields with identical label+section
- Applied on both AcroForm and AI scan paths

### Date field auto-formatting
- Updated `date-group` input handler to auto-insert slashes as user types digits (`0115` ŌåÆ `01/15`)
- Skips reformatting on `delete`/`backspace` (`e.inputType?.startsWith('delete')`) to avoid cursor fighting

### Retry/fail UI (production version)
Implemented the retry pip track and file card as real production UI (no demo gate):

**Retry pip track**
- 3 filled circles (`#78a9ff`) rendered inside `agent-brief-content` on first retry
- Pips fill progressively per attempt
- Countdown element below pips ticks independently in blue

**File card**
- Injected after `scan-brief` on first retry
- Shows filename, size, type; click opens file in new tab (blob URL revoked after 10s)
- Cleaned up via `cleanupRetryUI()` on resolve or total failure

**Rotating messages**
- Two tiers: optimistic (attempt 1) and heavy (attempt 2+)
- Messages cycle every 5s via `setInterval` + `typewrite()` without `await`
- 600ms delay before first typewrite

**Failure fallback**
- Updated "The AI service is overloaded" message to include "your document is still loaded"
